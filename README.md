# 🌙✨ Moonlight PvB - Roblox Script

一個功能強大的 Roblox 腳本，提供 PvB 商店種子預測和自動掛機功能。

## ✨ 功能特色

- 🎲 **種子預測系統** - 預測 PvB 商店未來種子
- 🤖 **自動掛機系統** - 可自訂間隔的自動跳躍
- 📱 **月亮圖標介面** - 美觀的黑色主題介面
- ⚡ **快捷鍵支持** - 快速開關功能
- 🎯 **拖曳介面** - 可自由移動的視窗

## 🚀 快速開始

### 方法一：直接加載
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/你的用戶名/Moonlight-PvB-Script/main/src/main.lua"))()
```

方法二：手動安裝

1. 下載 src/main.lua
2. 在執行器中運行

📋 系統要求

· 支援的 Roblox 執行器
· 基本網路連線

🎮 使用方法

1. 運行腳本
2. 點擊左上角月亮圖標開啟介面
3. 使用種子預測功能
4. 設定自動跳躍間隔
5. 開啟自動掛機

🔧 快捷鍵

· F1 - 開關主介面
· F2 - 開關自動跳躍

```
